Configure the mission and CPU specific cFE Application Parameter files

In the cFE source distribution, there are a few cFE Application configuration files. These files exist only as an example of how mission and CPU specific settings can be set for a suite of cFE Applications. There is no requirement by the cFE to have application configuration files.

cFE Application configuration files

File Description
build/cpu[x]/inc/app_msgids.h Common cFE Application Message IDs for a CPU.
build/cpu[x]/inc/app_perfids.h Common cFE Application Performance IDs for a CPU
build/inc/app_mission_cfg.h Mission-wide cFE Application parameters.
