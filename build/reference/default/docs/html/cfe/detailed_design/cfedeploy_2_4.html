Configure a Set of cFE Applications for Each CPU
Suggested Application Directory Tree

Once the cFE core is configured and ready to build, the cFE applications can be configured for each CPU directory in the build directory. Each CPU directory can have a different set of applications and configurations. The application source code can come from the cfe-apps directory, or the applications can be contained completely within the CPU directory. The cFE source distribution has a set of cFE applications in the cfe-apps directory and a set of application directories and makefiles in each CPU directory (cpu1, cpu2, and cpu3).
