Load the cFE Core Executable on the Target

On desktop targets the cFE Core can be run right from the exe directory where it was built/installed. On embedded targets, the cFE Core has to be loaded into a remote file system, or booted over the network. On the RAD750, the cFE Core must be loaded into the EEPROM disk after the vxWorks kernel is booted. See the target specific sections for details on each platform.
