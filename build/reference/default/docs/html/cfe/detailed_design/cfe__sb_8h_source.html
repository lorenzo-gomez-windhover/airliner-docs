1187 /******************************************************************************
1188 ** Function: CFE_SB_MessageStringGet()
1189 **
1190 ** Purpose:
1191 ** Copies a string out of a software bus message
1192 **
1193 ** Strings within software bus messages have a defined/fixed maximum length, and
1194 ** may not necessarily be null terminated within the message. This presents a possible
1195 ** issue when using the C library functions to copy strings out of a message.
1196 **
1197 ** This function should replace use of C library functions such as strcpy/strncpy
1198 ** when copying strings out of software bus messages to local storage buffers.
1199 **
1200 ** Up to [SourceMaxSize] or [DestMaxSize-1] (whichever is smaller) characters will be
1201 ** coped from the source buffer to the destination buffer, and a NUL termination
1202 ** character will be written to the destination buffer as the last character.
1203 **
1204 ** If the DefaultString pointer is non-NULL, it will be used in place of the source
1205 ** string if the source is an empty string. This is typically a string constant that
1206 ** comes from the platform configuration, allowing default values to be assumed for
1207 ** fields that are unspecified.
1208 **
1209 ** IMPORTANT - the default string, if specified, must be null terminated. This will
1210 ** be the case if a string literal is passed in (the typical/expected use case).
1211 **
1212 ** If the default is NULL, then only the source string will be copied, and the result
1213 ** will be an empty string if the source was empty.
1214 **
1215 ** If the destination buffer is too small to store the entire string, it will be
1216 ** truncated, but it will still be null terminated.
1217 **
1218 ** Arguments:
1219 ** DestStringPtr - pointer to destination buffer
1220 ** SourceStringPtr - pointer to source buffer (component of SB message definition)
1221 ** DefaultString - alternative string to use if the source is an empty string
1222 ** DestMaxSize - size of the destination storage buffer (must be at least 2)
1223 ** SourceMaxSize - size of the source buffer as defined by the message definition
1224 **
1225 ** Returns:
1226 ** Length of result - number of characters copied, not including the terminating NUL.
1227 ** This should match the value that would be returned by "strlen(DestStringPtr)"
1228 **
1229 */
1230 int32 CFE_SB_MessageStringGet(char *DestStringPtr, const char *SourceStringPtr, const char *DefaultString, uint32 DestMaxSize, uint32 SourceMaxSize);
1231 
1232 /******************************************************************************
1233 ** Function: CFE_SB_MessageStringSet()
1234 **
1235 ** Purpose:
1236 ** Copies a string into a software bus message
1237 **
1238 ** Strings within software bus messages have a defined/fixed maximum length, and
1239 ** may not necessarily be null terminated within the message. This presents a possible
1240 ** issue when using the C library functions to copy strings out of a message.
1241 **
1242 ** This performs a very similar function to "strncpy()" except that the sizes
1243 ** of _both_ buffers are passed in. Neither buffer is required to be null-terminated,
1244 ** but copying will stop after the first termination character is encountered.
1245 **
1246 ** If the destination buffer is not completely filled by the source data (such as if
1247 ** the supplied string was shorter than the allotted length) the destination buffer
1248 ** will be padded with NUL characters up to the size of the buffer, similar to what
1249 ** strncpy() does. This ensures that the entire destination buffer is set.
1250 **
1251 ** NOTE - if the source string buffer is already guaranteed to be null terminated,
1252 ** then there is no difference between the C library "strncpy()" function and this
1253 ** implementation. It is only necessary to use this when termination of the source
1254 ** buffer is not guaranteed.
1255 **
1256 ** Arguments:
1257 ** DestStringPtr - pointer to destination buffer
1258 ** SourceStringPtr - pointer to source buffer (component of SB message definition)
1259 ** DestMaxSize - size of the destination storage buffer (must be at least 2)
1260 ** SourceMaxSize - size of the source buffer as defined by the message definition
1261 **
1262 ** Returns:
1263 ** Length of result - number of actual data characters copied
1264 ** This should match the value that would be returned by "strlen(DestStringPtr)"
1265 **
1266 */
1267 int32 CFE_SB_MessageStringSet(char *DestStringPtr, const char *SourceStringPtr, uint32 DestMaxSize, uint32 SourceMaxSize);
