cFE Deployment Guide

The purpose of this document is to provide guidelines and conventions for the deployment of the cFE to a desired platform or platforms. In addition, instructions are included for creating a ground database and pages in order to provide a user interface. Note that the ground database is specific to the ASIST ground system.

IMPORTANT NOTE: The method for building the cFE in this guide is not recommended for use with CFS applications. For building the cFE and CFS applications, refer to the CFS Deployment Guide which is located in the CFS repository.
