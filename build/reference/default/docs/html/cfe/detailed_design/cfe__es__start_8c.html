uint32  StartSubtype,
uint32  ModeId,
const char *  StartFilePath 
)

cFE Main Entry Point used by Board Support Package to start cFE

Description
cFE main entry point. This is the entry point into the cFE software. It is called only by the Board Support Package software.
Assumptions, External Events, and Notes:
None
Parameters
[in]StartTypeIdentifies whether this was a #CFE_PSP_RST_TYPE_POWERON or #CFE_PSP_RST_TYPE_PROCESSOR.
[in]StartSubtypeSpecifies, in more detail, what caused the StartType identified above. See CFE_ES_POWER_CYCLE for possible examples.
[in]ModeIdIdentifies the source of the Boot as determined by the BSP.
[in]StartFilePathIdentifies the startup file to use to initialize the cFE apps.
