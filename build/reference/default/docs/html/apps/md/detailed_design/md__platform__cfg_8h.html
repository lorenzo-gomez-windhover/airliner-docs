#define MD_SIGNATURE_FIELD_LENGTH   32
Purpose:
Number of characters used in the Signature Field
Limits
Signature field length needs to be a multiple of 4 so that dwell packet is a multiple of 4 bytes and no compiler padding will occur. Note that the final character of the signature string must be a null character, so the effective length of user definable characters is one less than the defined length. The length specified must therefore be at least 4.
