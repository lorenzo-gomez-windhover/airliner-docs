Validate dwell entries in specified Dwell Table.

Validate memory ranges for dwell address and field length, and validate field length size for entries in specified Dwell Table.

int32 MD_TableValidationFunc ( void *  TblPtr)

Dwell Table Validation Function.

Description
This function indicates whether the Dwell Table corresponding to the input table pointer is valid.
Assumptions, External Events, and Notes:
This function gets registered with Table Services as a callback function for validating Dwell Tables that are loaded from the ground, and so must be in accordance with the call signature specified by Table Services. In addition, the function is used by Memory Dwell to validate Dwell Tables that have been recovered from a Critical Data Store.
Parameters
[in]TblPtrTable pointer
Returns
CFE_SUCCESS
MD_TBL_ENA_FLAG_ERROR Enable flag in table load is invalid (valid values are 0 and 1)
MD_ZERO_RATE_TBL_ERROR Table has zero value for total delay, and at least one dwell specified.
MD_RESOLVE_ERROR Symbolic address couldn't be resolved.
MD_INVALID_ADDR_ERROR Invalid address found.
MD_INVALID_LEN_ERROR Invalid dwell length found.
MD_NOT_ALIGNED_ERROR Dwell address improperly aligned for specified dwell length.
