void MD_DwellLoop ( void  )

Process Dwell Packets.

Description
Look at each table. If the table is enabled and its countdown counter times out, add dwell data to dwell packet until a packet delay is reached or the dwell packet is completed. If dwell packet is completed, send the packet and reset the counter.
Assumptions, External Events, and Notes:
None
Return values
None
