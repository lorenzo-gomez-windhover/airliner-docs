CFS Memory Dwell Configuration Parameters
CFS Memory Dwell Commands CFS Memory Dwell Telemetry CFS Memory Dwell Table Definitions

The following are configuration parameters used to configure the CFS Memory Dwell Application either for each platform or for a mission as a whole.

Global MD_DWELL_TABLE_SIZE
Maximum number of dwell specifications (address/delay/length) in a Dwell Table.
Global MD_ENFORCE_DWORD_ALIGN
Option of whether 32 bit integers must be aligned to 32 bit boundaries. 1 indicates 'yes' (32-bit boundary alignment enforced), 0 indicates 'no' (16-bit boundary alignment enforced).
Global MD_MISSION_REV
Mission specific version number for MD application
Global MD_NUM_DWELL_TABLES
Number of memory dwell tables.
Global MD_SIGNATURE_FIELD_LENGTH
Number of characters used in the Signature Field
Global MD_SIGNATURE_OPTION
Option of whether a signature field will be reserved in dwell packets. 1 indicates 'yes', 0 indicates 'no'.
Global MD_TBL_FILENAME_FORMAT
Memory Dwell Base Filename
