void LGC::AppMain ( void  )

Landing Gear Control (LGC) application entry point.

Description
Landing Gear Control Task application entry point. This function performs app initialization, then waits for the cFE ES Startup Sync, then executes the main processing loop.
Assumptions, External Events, and Notes:
