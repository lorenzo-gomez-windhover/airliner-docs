#define MAC_MAJOR_VERSION   (0)

Major version number for MAC application.

Description:
An application version number consists of four parts: major version number, minor version number, revision number and mission specific revision number. The mission specific revision number is defined in "ci_platform_cfg.h".
Limits:
Must be defined as a numeric value that is greater than or equal to zero.
