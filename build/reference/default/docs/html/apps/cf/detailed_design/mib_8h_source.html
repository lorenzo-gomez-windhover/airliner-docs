13  * BACKGROUND: The MIB is a database of CFDP configuration parameters.
14  * CFDP defines two classes of MIB parameters:
15  * "Local entity" parameters apply to all transactions (regardless of who
16  * the CFDP partner is), and there is only one value for each parameter.
17  * "Remote entity" parameters apply to one specific CFDP partner.
18  * Each of our CFDP partners may have a different setting for these
19  * parameters.
20  * SUMMARY: This module allows clients to both set and get the value of
21  * any MIB parameter. There is only one routine for setting an MIB
22  * parameter (see the first routine below); all the others are for
23  * getting the value of an MIB parameter.
24  * CHANGES:
25  * 2007_07_31 Tim Ray
26  * - Official release. (see pre_release_changes.log for earlier changes)
