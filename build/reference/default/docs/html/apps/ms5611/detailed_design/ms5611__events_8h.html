MS5611_RESERVED_EID

MS5611_INIT_INF_EID 

'Initialized. Version %d.%d.%d.%d'

Event Message:
'Initialized. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the MS5611 task has completed initialization.

The first d field contains the application major version defined in MS5611_MAJOR_VERSION. The second d field contains the application minor version defined in MS5611_MINOR_VERSION. The third d field contains the application revision number defined in MS5611_REVISION. The fourth d field contains the application revision number defined in MS5611_MISSION_REV.

MS5611_CMD_NOOP_EID 

'Recvd NOOP. Version %d.%d.%d.%d'

Event Message:
'Recvd NOOP. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the CFS MS5611 Task receives a NoOp command.

The first u field contains the application major version defined in MS5611_MAJOR_VERSION. The first d field contains the application major version defined in MS5611_MAJOR_VERSION. The second d field contains the application minor version defined in MS5611_MINOR_VERSION. The third d field contains the application revision number defined in MS5611_REVISION. The fourth d field contains the application revision number defined in MS5611_MISSION_REV.

MS5611_SUBSCRIBE_ERR_EID 

'%s Pipe failed to subscribe to %s. (0x%08X)'

Event Message:
'%s Pipe failed to subscribe to %s. (0x%08X)'
Type: ERROR
Cause:

This event message is issued when the CFS MS5611 Task fails to subscribe to a message at initialization.

The first s field contains the name of the pipe. The second s field contains the macro name of the message. The 32 bit hexadecimal number is the error code returned by CFE.
