Application Tailoring to the Mission

The px4lib_mission_cfg.h file, the px4lib_msgids.h, and the px4lib_perfids.h files are tailored for the mission. These are built into the application during compilation.

Application Tailoring to the Platform

In the px4lib_platform_cfg.h file, there are configuration parameters that control PX4LIB application behavior. This should be tailored to the specific platform needs. This behavior is built into the application during compilation.
