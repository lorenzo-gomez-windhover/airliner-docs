Detailed Description

Calculation / lookup table for Earth's magnetic field declination.

Lookup table from Scott Ferguson scott.nosp@m.from.nosp@m.scott.nosp@m.@gma.nosp@m.il.co.nosp@m.m and Stephan Brown steph.nosp@m.an.b.nosp@m.rown..nosp@m.07@g.nosp@m.mail..nosp@m.com

XXX Lookup table currently too coarse in resolution (only full degrees) and lat/lon res - needs extension medium term.
