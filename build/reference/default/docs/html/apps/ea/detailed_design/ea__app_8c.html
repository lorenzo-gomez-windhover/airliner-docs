void EA_AppMain ( void  )

CFS External Applications Task (EA) application entry point.

Description
CFS External Applications Task application entry point. This function performs app initialization, then waits for the cFE ES Startup Sync, then executes the RPR main processing loop.
Assumptions, External Events, and Notes:
If there is an unrecoverable failure during initialization the main loop is never executed and the application will exit.
