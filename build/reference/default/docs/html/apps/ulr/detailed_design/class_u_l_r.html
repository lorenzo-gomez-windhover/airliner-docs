void ULR::AppMain ( void  )

Aerotenna uLanding Radar (ULR) application entry point.

Description
Aerotenna uLanding Radar Task application entry point. This function performs app initialization, then waits for the cFE ES Startup Sync, then executes the main processing loop.
Assumptions, External Events, and Notes:
If there is an unrecoverable failure during initialization the main loop is never executed and the application will exit.
