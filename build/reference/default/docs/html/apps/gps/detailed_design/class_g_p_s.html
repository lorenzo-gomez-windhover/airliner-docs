void GPS::AppMain ( void  )

Global Positioning System (GPS) application entry point.

Description
Global Positioning System Task application entry point. This function performs app initialization, then waits for the cFE ES Startup Sync, then executes the main processing loop.
Assumptions, External Events, and Notes:
If there is an unrecoverable failure during initialization the main loop is never executed and the application will exit.
