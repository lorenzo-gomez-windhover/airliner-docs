void TO_Classifier_Run ( TO_ChannelData_t *  channel)

Runs the Classifier algorithm on incoming message flows.

Description
This runs a priority based classification algorithm that pops messages from incoming message flows, and copies the messages to the priority queue specified in the configuration table.
Assumptions, External Events, and Notes:
There is only one caller of this function and this caller enforces that the call will never contain a null pointer, therefore the null pointer condition is not checked.
Parameters
[in]channelA TO_ChannelData_t pointer that references the channel data structure
