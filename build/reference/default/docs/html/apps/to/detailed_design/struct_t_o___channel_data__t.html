TO_ChannelTbl_t* TO_ChannelData_t::ConfigTblPtr

Config Table Pointer.

A pointer to config table data from cFE table services.
Note, this pointer can be null if the table data is not current or valid. The reasons for this can occur outside of nominal app function and the pointer could be restored to a non-null value at another time.
