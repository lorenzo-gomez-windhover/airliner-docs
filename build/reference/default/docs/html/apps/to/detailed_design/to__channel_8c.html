const char *  CfCntSemName 
)

Configures and initializes a single channel.

Description
Initializes channel specific data and structures. Sets up channel pipe as well as initializing the cf throttling semaphore for each channel.
Assumptions, External Events, and Notes:
Semaphore name must be unique among channels; initialization will fail if two channels attempt to use the same semaphore. The pointers for ChannelName, ConfigTableName, ConfigTableFileName, DumpTableName, and CfCntSemName are not checked for NULL since those are passed in as arguments that have been macro defined in to_platform_cfg.h and are checked in a verify header to ensure the macros exist.

Only the index is range-checked and no other argument checking is performed.

Parameters
[in]indexIndex of the channel, cannot be greater than TO_MAX_CHANNELS
[in]ChannelNameName of channel, will be prefixed with TO_ . NOTE: Since each channel has a child task thread with the same name, and in RTEMS threads must have names of 4 chars, this param should be a single char.

The 4-char limit is an RTEMS limit. The RTEMS OSAL only takes the first 4 chars of the name per RTEMS limits for the underlying RTEMS message queue but still preserves the full name given to it in the OSAL The 1-char limit helps with keeping the name short enough to view it via RTEMS reports

Parameters
[in]ConfigTableNameName of the config table for the channel as referred to by TBL services
[in]ConfigTableFileNameFile path of the channel config table
[in]DumpTableNameName of the dump table for the channel as referred to by TBL services
[in]CfCntSemMaxThe initial maximum value of the CF throttling semaphore for this channel.
[in]CfCntSemNameName of the CF throttling semaphore, must be unique among each channel.
Returns
CFE_SUCCESS if successful. TO_CHANNEL_OUT_OF_RANGE_ERR or OSAL error if unsuccessful.
