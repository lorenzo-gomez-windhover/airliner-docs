uint16 TO_MessageFlow_t::MsgLimit

Maximum number of messages in a Message Flow.

Units: Number of messages that can be in a Message Flow.
Limits: The mininum value is 1. The maximum value depends on available memory and is an OPS driven parameter. The sum of all the msglimits in a priority queue should not be greater than the msglimit of that priority queue.
