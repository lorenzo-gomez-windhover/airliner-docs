Detailed Description

Supported multirotor geometries.

Values are generated by the multi_tables script and placed to mixer_multirotor.generated.h Multi-rotor mixer for pre-defined vehicle geometries.

Collects four inputs (roll, pitch, yaw, thrust) and mixes them to a set of outputs based on the configured geometry.
