enum VM_BatteryFailsafeMode_t

Battery failsafe modes. These define how VM reacts to battery low or failure.

Enumerator
VM_BATTERY_FAILSAFE_MODE_WARNING 

Publish a warning event.

VM_BATTERY_FAILSAFE_MODE_RETURN 

Return to Launch.

VM_BATTERY_FAILSAFE_MODE_LAND 

Land immediately.

VM_BATTERY_FAILSAFE_MODE_RETURN_IF_CRIT_LOW_LAND_IF_DANGER_LOW 

Return to Launch if low, but land immediately if even lower.
