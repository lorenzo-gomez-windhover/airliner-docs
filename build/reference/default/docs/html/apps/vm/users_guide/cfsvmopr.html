VM Application Operation

The following operations are carried out by the VM Application:

1. Set Home Position

a. Computes home latitutde, logitude, altitude and yaw from vehicle global position.
b. Sets new home position on arming.
c. Sets new home position on launch.

2. Update Navigation State

a. Updates naviagation state of vehicle status message based on ingested command.

3. Monitor Activities

a. Moniters the systems for red flags such as low battery, unarmed takeoffs, engine, gps faliure, rc loss.
b. Responds to red flags with appropriate reporting or contengent actions.

4. Send Housekeeping

a. On receipt of a "Send HK" command, VM publishes a housekeeping message to the software bus containing general health and status.
