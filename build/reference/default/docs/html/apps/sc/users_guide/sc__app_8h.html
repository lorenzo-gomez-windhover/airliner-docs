void SC_RegisterManageCmds ( void  )

Register to receive cFE Table Services manage request commands.

Description
This function provides cFE Table Services with the information necessary to send a notification command when one of the SC dump only tables has a dump pending, or when one of the SC loadable tables has a load pending. Upon receipt of the command, the command handler will call the cFE Table Services API function to manage the table. This sequence of events ensures that dump tables are not being updated by SC at the same moment that the dump occurs, and likewise, that loadable tables are not being referenced by SC at the moment that the update occurs.
Assumptions, External Events, and Notes:
None
Parameters
[in](none)
