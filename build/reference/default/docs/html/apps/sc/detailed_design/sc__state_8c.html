void SC_GetNextRtsCommand ( void  )

Gets the next RTS command to run.

Description
This routine is called when SC_ProcessRtpCommand executes an RTS command and needs to get the next command in the buffer. This routine will get the next RTS command from the currently executing RTS on the active RTP. If this routine finds a fatal error with fetching the next RTS command or cannot find a next RTS command, then the sequence and RTP is stopped.
Assumptions, External Events, and Notes:
None
