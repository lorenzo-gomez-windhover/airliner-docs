#define PE_SCH_PIPE_PEND_TIME   (2000)

The SB pend behavior type for the Scheduler pipe.

Limits:
One of: CFE_SB_POLL, CFE_SB_PEND_FOREVER, or the number of milliseconds to wait for a new message (recommended). Note, using CFE_SB_PEND_FOREVER may cause an unresponsive application if no messages arrive on this pipe.
