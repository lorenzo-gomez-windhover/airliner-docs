PE_RESERVED_EID 

Value of zero is reserved, and should not be used.

PE_INIT_INF_EID 

'Initialized. Version %d.%d.%d.%d'

Event Message:
'Initialized. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the PE task has completed initialization.

The first d field contains the application major version defined in PE_MAJOR_VERSION. The second d field contains the application minor version defined in PE_MINOR_VERSION. The third d field contains the application revision number defined in PE_REVISION. The fourth d field contains the application revision number defined in PE_MISSION_REV.

PE_CMD_NOOP_EID 

'Recvd NOOP. Version %d.%d.%d.%d'

Event Message:
'Recvd NOOP. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the CFS PE Task receives a NoOp command.

The first u field contains the application major version defined in PE_MAJOR_VERSION. The first d field contains the application major version defined in PE_MAJOR_VERSION. The second d field contains the application minor version defined in PE_MINOR_VERSION. The third d field contains the application revision number defined in PE_REVISION. The fourth d field contains the application revision number defined in PE_MISSION_REV.

PE_SUBSCRIBE_ERR_EID 

'%s Pipe failed to subscribe to %s. (0x%08X)'

Event Message:
'%s Pipe failed to subscribe to %s. (0x%08X)'
Type: ERROR
Cause:

This event message is issued when the CFS PE Task fails to subscribe to a message at initialization.

The first s field contains the name of the pipe. The second s field contains the macro name of the message. The 32 bit hexadecimal number is the error code returned by CFE.
