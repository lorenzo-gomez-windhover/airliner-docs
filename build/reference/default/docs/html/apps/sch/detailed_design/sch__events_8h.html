#define SCH_MINOR_FRAME_TIMER_ACC_WARN_EID   13

'OS Timer Accuracy (d > reqd d usec) requires Minor Frame MET sync'

Event Message:
'OS Timer Accuracy (d > reqd d usec) requires Minor Frame MET sync'
Type: INFORMATION
Cause:

This event message is generated when the minor frame timer obtained from OSAL has a specified clock accuracy that is not adequate enough for the Scheduler to maintain precise enough timing. The Scheduler will try to maintain conistent timing by implementing an additional minor frame synchronization to the MET clock. This means the Scheduler may end up needing to process more than one minor frame at a time if the minor frame timer does not stay close enough to required time.

The first d field specifies the clock accuracy (in microseconds) obtained for the minor frame timer and the second d field specifies the clock accuracy (in microseconds) that Scheduler has determined is necessary to maintain adequate timing.
