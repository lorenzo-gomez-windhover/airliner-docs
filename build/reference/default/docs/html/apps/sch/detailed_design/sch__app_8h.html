#define SCH_MAX_SYNC_ATTEMPTS   (SCH_TOTAL_SLOTS * 3)

Maximum number of minor frames to sample looking for subsecs = 0.

Maximum number of minor frame timer expirations to allow before giving up on finding the slot whose MET subseconds field is zero. It is assumed that three complete major frames should be sufficient for finding such a slot.
