void SCH_DisableProcessing ( void  )

Informs SCH Application that processing of schedule table should halt.

Description
This API increments the disable schedule table processing reference count. Each time this API is called must be matched with a similar call to the SCH_EnableProcessing API before the SCH Application will resume processing of the schedule table.
Assumptions, External Events, and Notes:
