CFS Scheduler Configuration Parameters
CFS Scheduler Commands CFS Scheduler Telemetry CFS Scheduler Table Definitions

The following are configuration parameters used to configure the CFS Scheduler Application either for each platform or for a mission as a whole.

Global SCH_ENTRIES_PER_SLOT
Maximum number of Activities per Minor Frame
Global SCH_LIB_DIS_CTR
Scheduler API Library Initial Inhibition Count
Global SCH_LIB_PRESENCE
Scheduler API Library Usage Status
Global SCH_MAX_LAG_COUNT
Maximum Number of slots allowed for catch-up before skipping
Global SCH_MAX_MESSAGES
Maximum Number of Message Definitions in Message Definition Table
Global SCH_MAX_MSG_WORDS
Maximum Length, in Words, of a Message
Global SCH_MAX_NOISY_MAJORF
Maximum Number of consecutive Noisy Major Frame signals before they are ignored
Global SCH_MAX_SLOTS_PER_WAKEUP
Maximum Number of Slots to be processed when in "Catch Up" mode
Global SCH_MDT_MAX_MSG_ID
Maximum Message ID allowed in Message Definition Table
Global SCH_MDT_MIN_MSG_ID
Minimum Message ID allowed in Message Definition Table
Global SCH_MESSAGE_FILENAME
Default SCH Message Definition Table Filename
Global SCH_MICROS_PER_MAJOR_FRAME
Major Frame Period (in microseconds)
Global SCH_MISSION_REV
Mission specific version number for SCH application
Global SCH_PIPE_DEPTH
Software Bus Command Pipe Depth
Global SCH_SCHEDULE_FILENAME
Default SCH Schedule Definition Table Filename
Global SCH_STARTUP_PERIOD
Time, in microseconds, to wait for first Major Frame Sync to arrive
Global SCH_STARTUP_SYNC_TIMEOUT
Time, in milliseconds, to wait for all applications to be started and ready to run
Global SCH_SYNC_SLOT_DRIFT_WINDOW
Additional time allowed in Sync Slot to wait for Major Frame Sync (in microseconds)
Global SCH_TOTAL_SLOTS
Minor Frame Frequency (in Hz)
