void HS_MonitorUtilization ( void  )

Monitor the utilization tracker counter.

Description
Monitors the utilization tracker counter incremented by the Idle Task, converting it into an estimated CPU Utilization for the previous cycle. If the utilization is over a certain theshold for a certain amount of time, an event is output.
