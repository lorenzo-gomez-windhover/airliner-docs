Enumeration Type Documentation

enum CI_INGEST_BEHAVIOR

CI Ingest Mode.

Note
The various modes in which CI can run in. Optimistic means that CI will allow unregistered (unknown) commands it ingests that are valid to be published to the CFE SB. Commands that are registered will adhere to the stepping they are registered with. Pessimistic mode will reject any command ingested that is not registered.
