void LC_SampleAPs ( uint16  StartIndex,
uint16  EndIndex 
)

Sample actionpoints.

Description
Support function for LC_SampleAPReq that will sample the selected actionpoints. The start and end arguments define which actionpoint(s) the command will sample. If both the start and end arguments are set to LC_ALL_ACTIONPOINTS, the command will be interpreted as a request to sample all actionpoints (heritage). Otherwise, the start index must be less than or equal to the end index, and both must be within the bounds of the actionpoint table.
Assumptions, External Events, and Notes:
None
Parameters
[in]StartIndexThe first actionpoint to sample (zero based actionpoint table index)
[in]EndIndexThe last actionpoint to sample (zero based actionpoint table index)
