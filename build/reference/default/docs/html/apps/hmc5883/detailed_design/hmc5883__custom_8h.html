Custom function to initialize custom events.

Description
This function is called in init event before CFE_EVS_Register to add custom events to the event filter table.
Assumptions, External Events, and Notes:
This function must be defined, but not all custom layers will do anything in this function.
Parameters
[in]indThe current event table index.
[out]EventTblThe current event table to modify.
Returns
number of events written to the filter table and -1 for failure i.e. CFE_EVS_MAX_EVENT_FILTERS reached.
