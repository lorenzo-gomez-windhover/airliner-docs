#define HMC5883_MAJOR_VERSION   (1)

Major version number for HMC5883 application.

Description:
An application version number consists of four parts: major version number, minor version number, revision number and mission specific revision number. The mission specific revision number is defined in "hmc5883_platform_cfg.h".
Limits:
Must be defined as a numeric value that is greater than or equal to zero.
