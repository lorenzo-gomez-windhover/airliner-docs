578 ** the hash table. The indexed hash table entry contains a
579 ** linked list where each link identifies a single entry in
580 ** the packet filter table. Thus, the scope of searching the

583 ** table by a factor equal to the size of the hash table.

585 ** \par Called From:
586 ** - Hash table creation function (after load filter table)
587 ** - Find messageID in filter table function
588 **
589 ** \par Assumptions, External Events, and Notes:
590 ** (none)
591 **

593 ** \param [out] Index of hash table entry for input message ID
594 **
595 ** \sa #DS_HashLink_t, #DS_TableCreateHash, #DS_TableFindMsgID
