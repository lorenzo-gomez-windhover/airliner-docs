int32 CS_ComputeApp ( CS_Res_App_Table_Entry_t *  ResultsEntry,
uint32 *  ComputedCSValue,
boolean *  DoneWithEntry 
)

Computes checksums on applications.

Description
Computes checksums up to MaxBytesPerCycle bytes every call. This function is used to compute checksums for applications.
Assumptions, External Events, and Notes:
None
Parameters
[in]*ResultsEntryA pointer to the entry in a table that we want to compute the checksum on.
[in]ComputedCSValueA pointer to the computed checksum that will be assigned the checksum of the entry.
[in]DoneWithEntryA pointer to a boolean that will be assigned a value based on whether or not the specified entry's checksum was completed.
[out]*ComputedCSValueValue used to determine the computed checksum, if completed
[out]*DoneWithEntryValue that specifies whether or not the specified entry's checksum was completed during this call.
Returns
CS_SUCCESS Success return code when a checksum compare did not fail.
CS_ERROR Error code returned when a checksum compare failed.
CS_ERR_NOT_FOUND Error code returned on a Table or App operation where the app or table requested could not be found.
