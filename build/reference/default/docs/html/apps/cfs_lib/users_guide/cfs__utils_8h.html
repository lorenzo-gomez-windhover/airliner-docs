uint32  TypeCRC 
)

Compute CRC from a file.

Description
This function will compute the cyclic redundancy check (CRC) value of data in a file
Assumptions, External Events, and Notes:
The computation starts from the current location of the file pointer to the end of file
Parameters
[in]FileHandleThe open file handle of the file to scan
[in]CrcPtrWhere to store the computed CRC.
[in]TypeCRCOne of the enumerated CRC types supported by the CFE_ES_CalculateCRC function
[out]*CrcPtrThe computed CRC. Only updated if the return value is OS_SUCCESS
Returns
Returns OS_SUCCESS if no error
The return code from OS_read if an error occured
See also
CFE_ES_CalculateCRC, OS_read

Determine whether Software Bus message packet is filtered.

Description
This routine will apply the CFS filter algorithm to the packet to determine whether the packet should be filtered. This same algorithm may be used by multiple applications (DS, TO, etc.)
Assumptions, External Events, and Notes:
None
Parameters
[in]MessagePtrPointer to a Software Bus message packet
[in]FilterTypePacket sequence count (1) or timestamp (2)
[in]Algorithm_NAlgorithm parameter N "pass this many"
[in]Algorithm_XAlgorithm parameter X "out of this many"
[in]Algorithm_OAlgorithm parameter O "at this offset"
Returns
Returns TRUE if the packet should be filtered (not used)
Returns FALSE if the packet should not be filtered (used)
See also
CFS_IsPacketFiltered

Resolve symbolic address.

Description
This routine will resolve a symbol name and optional address offset to an absolute address
Assumptions, External Events, and Notes:
If the symbol name is a NUL (empty) string, then the offset becomes the absolute address
Parameters
[in]SymAddrA CFS_SymAddr_t pointer that holds the symbol name and optional offset
[in]ResolvedAddrA pointer to the location to store the resolved address in
[out]*ResolvedAddrThe fully resolved address. Only valid if the return value is TRUE
Returns
Returns TRUE if the resolution was successful
Returns FALSE if the resolution was not successful
