#define RCIN_MAJOR_VERSION   (1)

Major version number for RCIN application.

Description:
An application version number consists of four parts: major version number, minor version number, revision number and mission specific revision number. The mission specific revision number is defined in "rcin_platform_cfg.h".
Limits:
Must be defined as a numeric value that is greater than or equal to zero.
