enum RCIN_EventIds_t
Enumerator
RCIN_RESERVED_EID 

Value of zero is reserved, and should not be used.

RCIN_INIT_INF_EID 

'Initialized. Version %d.%d.%d.%d'

Event Message:
'Initialized. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the RCIN task has completed initialization.

The first d field contains the application major version defined in #RCIN_MAJOR_VERSION. The second d field contains the application minor version defined in #RCIN_MINOR_VERSION. The third d field contains the application revision number defined in #RCIN_REVISION. The fourth d field contains the application revision number defined in RCIN_MISSION_REV.

RCIN_CMD_NOOP_EID 

'Recvd NOOP. Version %d.%d.%d.%d'

Event Message:
'Recvd NOOP. Version %d.%d.%d.%d'
Type: INFORMATION
Cause:

This event message is issued when the CFS RCIN Task receives a NoOp command.

The first u field contains the application major version defined in #RCIN_MAJOR_VERSION. The first d field contains the application major version defined in #RCIN_MAJOR_VERSION. The second d field contains the application minor version defined in #RCIN_MINOR_VERSION. The third d field contains the application revision number defined in #RCIN_REVISION. The fourth d field contains the application revision number defined in RCIN_MISSION_REV.

RCIN_SUBSCRIBE_ERR_EID 

'%s Pipe failed to subscribe to %s. (0x%08X)'

Event Message:
'%s Pipe failed to subscribe to %s. (0x%08X)'
Type: ERROR
Cause:

This event message is issued when the CFS RCIN Task fails to subscribe to a message at initialization.

The first s field contains the name of the pipe. The second s field contains the macro name of the message. The 32 bit hexadecimal number is the error code returned by CFE.

RCIN_PIPE_INIT_ERR_EID 

'Failed to create %s pipe (0x%08X)'

Event Message:
'Failed to create %s pipe (0x%08X'
Type: ERROR
Cause:

This event message is issued when the CFS RCIN Task fails to create a CFE Software Bus pipe at initialization.

The s field contains the name of the pipe. The 32 bit hexadecimal number is the error code returned by CFE.

RCIN_CFGTBL_MANAGE_ERR_EID 

'Failed to manage Config table (0x%08X)'

Event Message:
'Failed to manage Config table (0x%08X)'
Type: ERROR
Cause:

The 32 bit hexadecimal number is the error code returned by CFE CFE_TBL_Manage function.

RCIN_CFGTBL_GETADDR_ERR_EID 

'Failed to get Config table's address (0x%08X)'

Event Message:
'Failed to get Config table's address (0x%08X)'
Type: ERROR
Cause:

The 32 bit hexadecimal number is the error code returned by CFE CFE_TBL_GetAddress function.

RCIN_RCVMSG_ERR_EID 

'%s pipe read error (0x%08X).'

Event Message:
'%s pipe read error (0x%08X).'
Type: ERROR
Cause:

This event message is issued when the CFS RCIN Task has had an error reading from a pipe.

The s field contains the name of the pipe. The 32 bit hexadecimal number is the error code returned by CFE.

RCIN_MSGID_ERR_EID 

'Recvd invalid %s msgId (0x%04X)'

Event Message:
'Recvd invalid %s msgId (0x%04X)'
Type: ERROR
Cause:

This event message is issued when the CFS RCIN Task has received an invalid message ID.

The s field contains the name of the pipe. The 16 bit hexadecimal number is the actual message ID received.

RCIN_CC_ERR_EID 

'Recvd invalid command code (%u)'

Event Message:
'Recvd invalid command code (%u)'
Type: ERROR
Cause:

This event message is issued when the CFS RCIN Task has received an invalid message ID.

The u field contains the actual command code received.

RCIN_MSGLEN_ERR_EID 

'Rcvd invalid msgLen: msgId=0x%08X, cmdCode=%d, msgLen=%d, expectedLen=%d"

Event Message:
'Rcvd invalid msgLen: msgId=0x%08X, cmdCode=%d, msgLen=%d, expectedLen=%d"
Type: ERROR
Cause:

This event message is issued when the CFS RCINTask has received a message with an invalid length.

The first d field contains the message ID. The second d field contains the command code. The third d field contains the actual length. The fourth d field contains the expected length.
