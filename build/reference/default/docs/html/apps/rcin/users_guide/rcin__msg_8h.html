#define RCIN_NOOP_CC   (0)
Name:
Noop
Description
Implements the Noop command that demonstrates the RCIN task is alive
Command Mnemonic(s):

Command Structure
RCIN_NoArgCmd_t
Command Verification
Successful execution of this command may be verified with the following telemetry:
  • $sc_$cpu_RCIN_CMDACTPCNT - command counter will increment
  • The #RCIN_CMD_INF_EID informational event message will be generated when the command is received
Error Conditions
This command may fail for the following reason(s):
